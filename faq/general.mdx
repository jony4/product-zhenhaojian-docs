# 常见问题

## 基础与使用

<AccordionGroup>
  <Accordion title="我是剪辑新手，容易上手吗？">
    非常容易！帧好剪专为非专业人士设计，界面直观，功能强大。我们提供丰富的教程和模板，您只需几分钟即可制作出专业级视频。
  </Accordion>
  <Accordion title="支持哪些设备使用？">
    帧好剪支持网页端（Windows/Mac/Linux 均可访问）以及 iOS 和 Android 移动端 App。您的项目会在云端自动同步，随时随地接续创作。
  </Accordion>
  <Accordion title="我的数据和视频内容安全吗？">
    非常安全。我们使用企业级加密技术保护您的数据传输和存储。您的视频内容仅您可见（除非您主动分享），我们绝不会将其用于其他商业用途。
  </Accordion>
</AccordionGroup>

## 功能特性

<AccordionGroup>
  <Accordion title="AI 自动剪辑功能是如何工作的？">
    我们的 AI 引擎会分析您的视频素材，自动识别精彩片段、语音内容和画面节奏，然后根据您选择的风格模板进行智能拼接、配乐和卡点，极大节省您的时间。
  </Accordion>
  <Accordion title="生成的素材和音乐有版权吗？">
    是的，我们内置的音乐库和素材库均已获得正版授权。专业版用户可以放心商用，无需担心版权纠纷。
  </Accordion>
  <Accordion title="支持导出哪些视频格式？">
    支持导出 MP4, MOV, GIF 等主流格式。分辨率最高支持 4K 60帧，满足各种平台的高清发布需求。
  </Accordion>
  <Accordion title="导出的视频有水印吗？">
    基础版（免费版）导出的视频会带有“帧好剪”的水印。升级到专业版或团队版后，即可享受无水印导出，并支持自定义品牌水印。
  </Accordion>
  <Accordion title="云端存储空间有限制吗？">
    免费用户拥有 2GB 云存储空间。专业版用户拥有 100GB 空间，团队版用户则享有 1TB 共享空间。如果需要更多，可以单独购买存储包。
  </Accordion>
  <Accordion title="可以多人协作编辑同一个项目吗？">
    可以。团队版支持多人实时协作，成员可以对同一项目进行批注、修改和审核，大幅提升团队工作效率。
  </Accordion>
   <Accordion title="是否提供 API 接口供开发者集成？">
    我们提供开放的 RESTful API，允许开发者将帧好剪的核心剪辑能力集成到自己的应用或工作流中。详情请查看开发者文档。
  </Accordion>
</AccordionGroup>

<AccordionGroup>

</AccordionGroup>
