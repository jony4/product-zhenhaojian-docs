# 场景示例

本目录包含帧好剪各类应用场景的详细制作指南和最佳实践。

## 文件列表

| 文件 | 场景 | 说明 |
|------|------|------|
| `corporate-video.mdx` | 企业宣传片 | 公司介绍、产品发布、企业文化展示等 |
| `product-showcase.mdx` | 产品展示视频 | 电商产品、品牌营销、产品发布等 |
| `social-media.mdx` | 社交媒体内容 | 抖音、快手、小红书、视频号等平台内容 |
| `education.mdx` | 教育培训视频 | 在线课程、知识讲解、技能培训等 |

## 场景分类

### 营销推广
适用于企业营销、品牌推广、产品宣传等商业场景。

- **企业宣传片**：展示公司形象和企业文化
- **产品展示视频**：突出产品特点和卖点

### 内容创作
适用于内容创作者、自媒体、教育工作者等。

- **社交媒体内容**：适配各大短视频平台的内容制作
- **教育培训视频**：知识传播和技能培训

## 每个场景包含的内容

1. **场景概述**：适用场景和目标用户
2. **制作流程**：从素材准备到发布的完整流程
3. **AI 工具应用**：如何使用帧好剪的 AI 能力
4. **最佳实践**：行业经验和制作技巧
5. **平台优化**：针对不同平台的优化建议
6. **常见问题**：制作过程中的常见问题解答

## 使用建议

1. **选择合适的场景**：根据您的需求选择对应的场景指南
2. **了解完整流程**：阅读制作流程，了解整体步骤
3. **学习最佳实践**：参考行业经验，提升视频质量
4. **灵活应用工具**：根据实际需求选择合适的 AI 工具
5. **持续优化**：根据数据反馈不断优化内容
