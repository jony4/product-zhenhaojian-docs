# 快速开始

欢迎使用帧好剪！本指南将帮助您快速熟悉平台的核心功能。

## 1. 注册账号

访问 [控制台](https://video.cutcut.video) 并使用手机号注册。

## 2. 选择工具

在首页选择您需要的工具，例如“多图解说”或“视频翻译”。

## 3. 上传素材

根据提示上传图片或视频素材。

## 4. 获取结果

等待 AI 处理完成，即可下载或分享您的作品。
