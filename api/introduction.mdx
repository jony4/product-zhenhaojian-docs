# API 概览

帧好剪提供强大的 RESTful API，允许开发者将视频处理、字幕翻译等能力集成到自己的应用程序中。

## 鉴权

所有 API 请求都需要在 Header 中包含 API Key：

```bash
Authorization: Bearer YOUR_API_KEY
```

## 核心接口

- `POST /v1/video/translate`: 视频翻译
- `POST /v1/subtitle/extract`: 字幕提取
- `POST /v1/images/commentary`: 多图解说

[查看完整 API 文档](https://docs.cutcut.video/api)
